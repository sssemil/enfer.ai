Problem

Search is not ground truth.

Search gives agents more context, but not necessarily better context. A retrieved page can be relevant and still be old, biased, promotional, copied, or wrong. Once that enters the model window, the agent reasons on it like truth.

Layer

Context should be checked before it is trusted.

  • Freshness. Is the source current enough for the task?
  • Source quality. Is it primary, credible, promotional, thin, or copied?
  • Claim support. Can the important claims be traced back to real evidence?
  • Contradiction risk. Does a better source disagree?
